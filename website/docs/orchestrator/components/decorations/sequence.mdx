# Les séquences et sous-séquences

Une séquence est une partie ou module de questionnaire.  
Une sous-séquence est une partie d'une séquence : un questionnaire comporte une ou plusieurs séquences, chacune d'elles peut comporter entre 0 et plusieurs sous-séquences. La sous-séquence regroupe des questions relatives à une même thématique.

Le composant Sequence (resp. Subsequence) permet d'afficher sur la page le titre ainsi que les éventuelles déclarations liées à la séquence (resp. sous-séquence).
