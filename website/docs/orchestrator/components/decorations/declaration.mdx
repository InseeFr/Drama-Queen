# Les déclarations

La déclaration ou aide est un texte qui aidera l'enquêté à comprendre la question et comment y répondre.

La mise en forme des déclarations dépend de leur type :

- les aides et instructions, en noir
- les consignes, en bleu
- les cartes-codes : _à venir_
