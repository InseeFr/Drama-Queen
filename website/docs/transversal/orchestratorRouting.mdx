# Renvoi vers l'orchestrateur

A l'ouverture d'un questionnaire, quel que soit le mode (collecte, relecture, visualisation), l'application redirige vers une version précise de l'orchestrateur. En s'appuyant sur la variable `lunaticModelVersion` du modèle de questionnaire, la règle actuelle est :

- par défaut : renvoi vers la version courrante de l'orchestrateur
- si `lunaticModelVersion` = '2.2.10' : renvoi vers "Queen-v1" avec une version 0.x, utilisant une version 1.x de Lunatic
