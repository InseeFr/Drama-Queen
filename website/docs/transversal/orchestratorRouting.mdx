# Renvoi vers l'orchestrateur

A l'ouverture d'un questionnaire, quel que soit le mode (collecte, relecture, visualisation), l'application redirige vers l'orchestrateur si la variable `lunaticModelVersion` du modèle de questionnaire a une version supérieure à `2.2.10`.
Si cette version est plus ancienne, le questionnaire n'est pas compatible avec l'application, et l'utilisateur est redirigé sur une page d'erreur.
