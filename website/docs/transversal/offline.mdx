# Fonctionnement hors ligne

L'application Drama-Queen est prévue pour pouvoir fonctionner seule, ou pour être embarquée directement dans une autre application.

## Drama-Queen "seul"

Lorsque l'application Drama-Queen est lancée directement sans application parent, elle ne permet pas un fonctionnement hors ligne.

## Drama-Queen embarqué dans une application parent

Lorsque l'application Drama-Queen est lancée depuis une application parent, elle permet un fonctionnement hors ligne.
