# Champ Texte

Le composant Input permet d'afficher un simple champ texte, caractérisé par une longueur maximale du texte.
Un controle supplémentaire est à ajouter pour définir une longueur précise.

Le composant TextArea permet d'afficher un champ texte plus long que l'Input.

En pied de page, des boutons "Ne sait pas" et "Refus" permettent d'indiquer une non réponse et ainsi poursuivre le déroulement du questionnaire.
La sélection d'une modalité de non réponse remet à vide le champ de réponse, et provoque le passage automatique à la page suivante.
En revenant sur la page, si une modalité de non réponse était sélectionnée, la saisie d'une réponse dans le champ déselectionne la modalité de non réponse.
