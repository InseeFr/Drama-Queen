# Pagination

Une seule question est affichée par page.

Les composants de séquence/sous-séquence contiennent également chacun une page associée.
Les pages ne contenant qu'une nouvelle sous-séquence sans déclaration ne sont pas affichées.

Dans la barre de navigation sur la droite de la page, on retrouve le `n° de page actuel / nombre de pages du questionnaire`.

Lorsque l'on se trouve à l'intérieur d'une boucle, le n° de page reste identique : on boucle sur des sous-pages, pour chaque itération.
On retouve alors un indicateur supplémentaire, semblable au précédent : `n° de sous-page actuel / nombre de sous-pages dans la boucle`.
